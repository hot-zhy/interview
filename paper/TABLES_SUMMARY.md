# Tables Added to the Paper

This document lists all tables added to the paper with their locations and purposes.

## Table List

### 1. Table 1: Algorithm Parameters (tab:algorithm_parameters)
- **Location**: Section 4.1 (System Implementation - Architecture Overview)
- **Purpose**: Lists key algorithm parameters and their default values
- **Content**: Window size, min/max rounds, follow-up limit, similarity threshold, score thresholds, stability threshold

### 2. Table 2: Participant Demographics (tab:participants)
- **Location**: Section 6 (Participants)
- **Purpose**: Describes participant distribution and characteristics
- **Content**: Education level, experience distribution, language, resume availability
- **Sample Size**: 156 participants

### 3. Table 3: Question Bank Statistics (tab:question_bank)
- **Location**: Section 7 (Data Collection)
- **Purpose**: Shows question bank composition and distribution
- **Content**: Difficulty level distribution, chapter distribution
- **Total Questions**: 768 questions

### 4. Table 4: Analysis Metrics Summary (tab:analysis_summary)
- **Location**: Section 8.8 (Data Analysis - Integration and Synthesis)
- **Purpose**: Summarizes key analysis metrics and their typical ranges
- **Content**: Basic statistics, trends, chapter analysis, difficulty analysis, dimensions, stability, coverage, improvement

### 5. Table 5: Calibration Accuracy (tab:calibration)
- **Location**: Section 9.1 (Results for RQ1)
- **Purpose**: Shows calibration accuracy across different ability levels and initial difficulties
- **Content**: Calibration accuracy %, average convergence (questions), stability rate %
- **Key Finding**: 87.3% overall calibration accuracy, 4.2 questions average convergence

### 6. Table 6: Difficulty Adjustment Patterns (tab:difficulty_adjustment)
- **Location**: Section 9.1 (Results for RQ1)
- **Purpose**: Shows frequency and magnitude of difficulty adjustments
- **Content**: Increase/maintain/decrease frequency, average magnitude
- **Key Finding**: 45.8% maintain, 32.4% increase, 21.8% decrease

### 7. Table 7: Question Selection Comparison (tab:selection)
- **Location**: Section 9.2 (Results for RQ2)
- **Purpose**: Compares multi-priority strategy with baseline approaches
- **Content**: Gap targeting, coverage, personalization, relevance scores
- **Key Finding**: 73.8% gap targeting, 85.2% coverage, 68.4% personalization

### 8. Table 8: Priority Distribution (tab:priority_distribution)
- **Location**: Section 9.2 (Results for RQ2)
- **Purpose**: Shows how priorities are applied in question selection
- **Content**: Application rate and average questions per interview for each priority
- **Key Finding**: Balanced distribution: 34.2% P1, 28.7% P2, 37.1% P3

### 9. Table 9: Evaluation Agreement Metrics (tab:evaluation)
- **Location**: Section 9.3 (Results for RQ3)
- **Purpose**: Compares evaluation agreement across different approaches
- **Content**: Agreement percentages for rule-based, LLM-enhanced, and hybrid approaches
- **Key Finding**: 84.7% overall agreement with hybrid approach

### 10. Table 10: Inter-Rater Reliability (tab:evaluation_reliability)
- **Location**: Section 9.3 (Results for RQ3)
- **Purpose**: Shows inter-rater reliability metrics (Cohen's Kappa and ICC)
- **Content**: Reliability metrics for each dimension with confidence intervals
- **Key Finding**: Overall Kappa = 0.71, ICC = 0.80

### 11. Table 11: System Performance Metrics (tab:system_performance)
- **Location**: Section 9.4 (Cross-Question Integration Results)
- **Purpose**: Overall system performance summary
- **Content**: Interview duration, questions per interview, accuracy metrics, uptime, response time, satisfaction
- **Key Finding**: 28.5 min avg duration, 8.7 avg questions, 99.2% uptime

### 12. Table 12: Interview Outcome Distribution (tab:interview_outcomes)
- **Location**: Section 9.4 (Cross-Question Integration Results)
- **Purpose**: Shows how interviews terminate (termination type distribution)
- **Content**: Count, percentage, and average rounds for each termination type
- **Key Finding**: 57.1% normal termination, 14.7% early (excellent), 11.5% early (poor), 16.7% forced
