# UI Screenshots for Paper

Place the following screenshot files in this directory to include them in the paper:

| File | Description |
|------|-------------|
| `ui_home.png` | Home page with navigation and Chinese/English language switcher |
| `ui_interview.png` | Interview page with avatar, chat area, and real-time expression video |
| `ui_report.png` | Report page with scores, strengths, weaknesses, and learning plan |

## How to Capture

1. Run the application: `streamlit run app/streamlit_app.py`
2. Log in and navigate to each page
3. Take screenshots (e.g., Windows: Win+Shift+S; Mac: Cmd+Shift+4)
4. Save as PNG in this folder with the names above

If the files are missing, the paper will show placeholder boxes with instructions.
